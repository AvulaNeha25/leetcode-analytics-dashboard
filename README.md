# 🚀 LeetCode Analytics Dashboard

A web-based dashboard that analyzes a user's LeetCode problem-solving progress.

This project allows users to track their LeetCode statistics, visualize difficulty distribution, and compare problem-solving performance between two users.

---

## 📊 Features

- Track LeetCode statistics by entering a username
- View total problems solved
- Difficulty breakdown (Easy / Medium / Hard)
- Visual charts for problem distribution
- Progress bars for difficulty levels
- Compare two LeetCode users
- Error handling for invalid usernames
- Clean dashboard UI

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript
- Chart.js

**Backend**
- Node.js
- Express.js

---

## ⚙️ How It Works

1. User enters a LeetCode username
2. Frontend sends a request to the backend server
3. Backend fetches user statistics from LeetCode GraphQL API
4. Data is processed and returned to the frontend
5. Dashboard displays statistics and charts
